## Request

- A Tailwind CSS based colour scheme, encompassing both light and dark themes.

## Requirements

- The colour scheme should be restrained and 'classical'.
- The light version should emulate the following
  [example](https://www.londonreviewbookshop.co.uk/events).
- The light version should use blue as the primary colour, rather than the green shown in
  the example.
- The dark version should be similar to the dark version of
  [React Hook Form](https://react-hook-form.com/) website.

## Deliverables

- Generate the Tailwind based CSS
- Generate an image of an example web page using the generated CSS.
- Create a tar.gz archive containing both the generated CSS and the image.
- The archive should be called 'colour_scheme.tar.gz'.
